simple git commands git init: initalises an empty git repository in the folder destination git add : adds a file of your specification into the git repository, into the staging area git add . : adds all files in the location inot your git repository git commit -m: commits the file to the local repository git status: finds out information regarding what files are modified and what files are there in the staging area. git log: prints out all the commits which have been done up until now. git branch : creates a new branch for the repository. get checkout : changes the editing from the current branch to the newly chosen branch. Commits will now be done in this new branch. git merge: merges 2 branches together, updating any commitments in the process. git remote add origin : adds the URL of the online repository to the local repository. git push -u origin master: Pushes the code from the master branch in the local repository to the master branch in the remote repository. git pull origin master: Pull the latest changes from the remote repository into the local repository into the local repository. git clone : clones an existing remote repository into your computer.
